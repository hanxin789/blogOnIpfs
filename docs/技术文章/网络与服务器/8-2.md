::: tip 开始是想用ubuntu做个测试服务器

用的是VMware15版本，安装好之后提示vm-tools工具是已经安装。但还是无法黏贴宿主机复制过去的信息。

:::

::: warning

解决方法：直接重新安装vm-tools

```shell
sudo apt autoremove vm-tools

sudo apt install vm-tools

sudo apt install vm-tools-desktop

reboot

```

:::
